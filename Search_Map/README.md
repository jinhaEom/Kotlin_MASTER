# 위치검색 지도 어플
- Retrofit,GSON, Coroutines 및 TMAP 지도 API 를 활용하여 검색화면 구성
- GoogleMap API를 활용하여 지도위치 화면 구성.
- POI Geo Reverse, Retrofit, GSON 를 활용하여 현재 내 위치 확인 기능 구성.
