## 로또번호 랜덤생성 어플
    - 기능 1. 자신이 원하는 숫자는 번호추가하기 버튼을 눌러 추가 할 수 있다.
    - 기능 2. 번호추가하기 후 남은자릿수는 자동생성시작 버튼을 눌러 랜덤값을 부여할 수있다.
    - (물론 맨처음부터 자동생성시작 버튼 클릭도 가능하다.)
