## Camera App
### CameraX 라이브러리 ( 카메라 앱 개발을 쉽게 할 수 있도록 만들어진 Jetpack 지원 라이브러리)
- 사용된 기술

    + 촬영 및 촬영 후 저장 구현하기
        - 캡쳐 버튼에 리스너 등록
        - 촬영하여 저장할 파일 선언
        - 사진 촬영 및 저장 콜백 구현
        - 사진 저장한 Uri를 Brodcast하여 다른 갤러리에 보이도록 설정
        - 저장한 사진에 대해 최신 사진의 미리보기 구현
    + Zoom In/Out rngus, Flash 구현
    + 찍은사진 확인 및 삭제기능 구현
