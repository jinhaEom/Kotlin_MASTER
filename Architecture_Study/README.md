아키텍쳐의 장점
 - 일관적인 코드작성
 - 생산성 향상
 - Test 용이성
 - 어플 개발의 방향성을 잡아줌.

일반적 아키텍쳐 종류
1. MVC : Model + View + Controller(activity,fragment 등등)
2. MVP : Model + View(ViewController) + Presenter(View에 넣어줄 데이터를 단순히 추상화)
3. MVVM : Model + View(Viewcontroller)+ ViewModel (단방향 형태의 제어)
4. MVVM + DataBinding
5. MVI : Model + View + Intent
6. MvRx, Flux, Ribs, 등등..
