## 계산기 어플
    - 사칙연산을 해주는 계산기 어플
    - 기능1. 연산 결과값을 = 버튼을 누르지 않아도 미리보기로 보여줌.
    - 기능2. db에 연산결과값을 넣어서 이전에 수행했던 연산을 볼 수 있음
    - 한계점1. : 정수연산만 가능하고 실수연산만 가능한점.
    - 한계점2. : 연산자를 한번만 사용할 수 있어 () 기능 구현 X
