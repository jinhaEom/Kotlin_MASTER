# 음악 스트리밍 어플
## 사용된 기술
Exoplayer 사용하기
- custom Controller
- playList 등
- Seekbar 커스텀
<br>
</br>

- Retrofit을 이용해 재생 목록을 받아와 구성.
- 재생목록을 클릭하여 ExoPlayer를 이용해 음악을 재생할 수 있음.
- 이전, 다음 트랙 버튼을 눌러서 이전, 다음 음악으로 재생하고 ui 업데이트
- PlayList화면과 Player화면 간의 전환 가능.
- Seebar를 커스텀하여 원하는 UI 로 표시 가능.
