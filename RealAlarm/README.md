## 알람 어플

+ 요구된 기술
    - AlarmManager - RealTime(실제 현재시간)으로 실행.
    - Notification - 알람이 울릴시간이 되었을시 notification 실행.
    - BackgroundReceiver - 알람앱을 켜놓지 않더라도 알람이 실행될수 있게함.
